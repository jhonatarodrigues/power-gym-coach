# Frontend Power Gym Coach

Base inicial do frontend mobile em Expo + React Native + TypeScript.

## Gerenciador de pacotes
- Padrao do projeto: `pnpm`

## Runtime
- Versao de Node esperada: `22`
- Antes de rodar o app:
  - `nvm use`

## Stack
- Expo
- React Native
- TypeScript
- Zustand
- React Navigation
- React Hook Form
- Storybook

## Estrutura
```text
frontend/
  src/
    components/
    forms/
    hooks/
    navigation/
    repository/
    screens/
    store/
    stories/
    theme/
    types/
```

## Direcao visual
- Fundo principal preto.
- Laranja como cor de acao e destaque.
- Aparencia forte e premium, sem visual generico.
